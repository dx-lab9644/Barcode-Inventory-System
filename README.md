# バーコード在庫管理システム

スマホでバーコードを読み取るだけの簡単在庫管理。Googleスプレッドシートベース。

## 特徴
- 📱 スマホで即導入
- 🇯🇵 日本語対応
- 💰 完全無料
- 📊 スプレッドシート自動記録
- ✏️ リスト機能（編集・削除）
- 🚀 一括登録

## 技術スタック
- Google Apps Script
- HTML5-QRCode
- Google Spreadsheet
